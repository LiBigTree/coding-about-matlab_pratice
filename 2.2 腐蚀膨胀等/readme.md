膨胀: 图像中亮的区域增长

> 使用卷积核B对图片A进行卷积运算，求局部最大值，这个卷积核可以有任意的形状和大小，通常是一个方形或者圆形。卷积核B通常有个锚点，通常位于卷积核的中央位置。
>
> 随着卷积核扫描图像，计算叠加区域的最大像素值，并将锚点的位置用最大值替换，从而导致图片中亮的区域增长。
